Display de elementos

En HTML tenemos elementos de línea, tenemos elementos de bloque y tenemos elementos de línea bloque

  • línea
  • bloque
  • línea-bloque

Elementos de línea

Los elementos de línea se presentan in-line o sea uno al lado del otro y estos de manera predeterminada no aceptan ancho ni altura
Sólo ocupan su propio espacio

soy un span soy un span soy un span

Elementos de bloque

Los elementos de bloque se presentan uno debajo del otro y estos de manera predeterminada si aceptan ancho ni altura
Ocupan todo el ancho del contenedor

soy un h3

soy un h3

soy un h3

Elementos de línea-bloque

Los elementos de línea-bloque se presentan uno al lado del otro pero estos si aceptan ancho y altura
